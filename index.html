Opcraft

The best way to manage your CraftBukkit server.

About

Opcraft is a little web app that helps you manage your CraftBukkit-powered Minecraft server. It has all the features you'd expect, including basic server management, user control, and item giving, as well as a real-time chat/console mode. It's free and open source, so you can even add your own awesome features.

Requirements

  • A Minecraft server (powered by CraftBukkit)
  • The JSONAPI Bukkit plugin
  • A great web browser. Google Chrome and Mobile Safari are my personal favorites, but anything that supports WebSockets should work great.
  • There is no step 3! (or whatever) If things aren't working, let me know.
